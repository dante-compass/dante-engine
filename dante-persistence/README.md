# 模块说明

常规系统，数据存储层默认只会使用一种数据存储介质。例如：数据库。对于一些有特殊性能需求的功能来说，默认数据存储介质未必能够满足，就需要更换为其它类型的数据存储介质。

在 Dante Cloud 中，Spring Authorization Server 是核心应用功能，也是访问使用量较高的功能，颁发 Token 和校验 Token（Opaque Token）均需要频繁使用。在数据库无法支持性能需求时，就需要更换其它的性能更优的存储介质。

`dante-persistence` 下的模块就为此目的设计。当前默认使用 JPA 对 Spring Authorization Server 等核心功能的数据存储进行实现，但已经提取了共性及抽象定义，如果需要使用其它的存储介质，参考 JPA 的结构和实现补充相关的实现代码模块即可。
